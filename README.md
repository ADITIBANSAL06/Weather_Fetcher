Weather Forecast Desktop App ☁️

A modern desktop weather application built using Python, Tkinter, and the OpenWeatherMap API.
This app allows users to check real-time weather information for any city through a clean graphical user interface.

Features 🚀

Real-time weather updates

GUI built with Tkinter

Fetches live weather using REST API

Displays:

Temperature

Humidity

Weather Condition

Wind Speed

Error handling for invalid city names

Beginner-friendly project structure

Technologies Used 🛠️

Python

Tkinter

Requests Library

OpenWeatherMap API

JSON
